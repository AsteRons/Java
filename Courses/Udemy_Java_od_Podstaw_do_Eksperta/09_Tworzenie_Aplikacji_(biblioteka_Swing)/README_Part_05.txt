- jeden z najlepszych Layoutów (pozwala wykonać praktycznie wszystko)


+ Grupujemy najpierw poziomo potem pionowo( na wzór tablicy dwuwymiarowej)



Horizontal - poziomo
równoległe - parallel
szeregowo (sekwencyjnie) - sequential


Vertical - pionowo

- stworzenie Layouta
- łączenie GRUPLAYOUT z szybka


Musimy określić pionowo oraz poziomo to samo inaczej wystąpią błędy


.addComponent(przycisk, x, y, z)
przycisk - nazwa przycisku

x - minimalna wielkosć naszego buttona  (Grouplayout.DEFAULT_SIZE)
y - wielkośc preferowana (jaka pojawi sie przy włączeni)
z - wielkosc maksymalna

Short.MAX_VALUE - wartosć nieskończona

+ rozszeżanie się wraz ze zwiększającym sie panelem

Rozszezanie się przycisku np. Anuluj wraz z krawedzią
