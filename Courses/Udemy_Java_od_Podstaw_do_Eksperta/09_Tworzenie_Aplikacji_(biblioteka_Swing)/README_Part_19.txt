PASEK MENU:

Program tworzy paski menu.

1. Tworzymy pasek menu
- setJMenuBar - ustawia najwyzej położony pasek
- addSeparator - oddziela paski w menu

Gdy:
JMenuBar - górny pasek z rozwijanymi opcjami
JMenu - opcja w którą można wchodzić
JMenuItem - opcja (przedmioty)

Można tu też używać:
- Java CheckBox 	( JCheckBoxMenuItem)
- Java RadioButton



Przydatne metody:

- setEnabled - możliwość zapisu
