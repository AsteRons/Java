LISTY KOMBINOWANE:

Program tworzy listy kombinowane. Po wybraniu jakiejś nazwy koloru w listach program zmienia kolor list na wybrany.

Lista kombinowana - lista ktore mają różne komponenty do wybrania
		- JComboBox
		
+ liste tworzymy podobnie jak przycisk
+ dodajemy elementy do listy za pomocą metody addItem()
+ program zmienia swoje tło poprzez rzutowanie
