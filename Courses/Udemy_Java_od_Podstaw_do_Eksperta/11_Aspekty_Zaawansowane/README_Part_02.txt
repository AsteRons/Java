ANIMACJA KROPELKI:

- Program animuje kropelki ( Jak wygaszcacz w Windowsie)

1. Stworzenie przycisku startującego
2. Stworzenie klasy z kropelkami
	+ tworzy obiekt kropelka 
	+ dodaje do niego obrazek
3. Stworzenie klasy do panelu animacji 
	+ Nadpisanie metody rysowania kropelek w osiach x i y tyle razy ile jest kropelek
	+ Stworzenie listy z elementami
	+ Dodanie kropelki
	+ Dodanie pętli for
		- stworzenie pętli for (wykjona sie tyle razy ile jest kropelek)
		- wykonanie metody ruszenia kropelki
		
Metoda rusz kropelke:

1. Pobiera granice panelu ( X, Y, Z, A)
2. Pisze warunek, zę jeśli kropelka dojdzie do granicy zmienia jego kierunek
