Parametry metod:

- można utworzyć metode, która zmienia dane z obiektu

Uwagi:
1. Metoda nie moze moddyfikować parametru podstawowego (będącego liczbą lub wart. logiczna)
	- jesli wykonuje dzialanie w metodzie i metoda nic nie zwraca wartość liczbowa sie nie zmienia

2. Metoda moze zmieniać stanu przekazywanego jako parametru
	- Metody mogą zmieniać stan parametrów będącego obiektami
	- działamy na obiektach
	
3. Metoda nie może zamienić obiektów
	- Metody nie mogą zamieniac dwóch obiektów
	- jesli wykonuje dzialanie w metodzie i metoda nic nie zwraca parametry obiektu sie nie zmienia
