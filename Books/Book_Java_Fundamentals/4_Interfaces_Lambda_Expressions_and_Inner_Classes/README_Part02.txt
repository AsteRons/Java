Interfejsy i wywołania zwrotne

- wywołanie zwrotne callback
- wykorzystanie biblioteki swing


Po uruchomieniu zegara wyświetlane jest okno dialogowe z przyciskiem OK zamykajacym okno.
W czasie oczekiwania na reakcię użytkownika co dziesięć sekund wyświetlane są aktualne data i godzina

Metody:
showMessageDialog - wyświetla okno dialogowe z komunikatem i przyciskiem OK. null - okno wyświetla sie na srodku ekranu

Timer - tworzy zegar powiadamiający  obieks listneer o upłyniętej liczbie ms

Toolkit getDefaultyToolkit() - zwraca standardowy zestaw narzędzi, na który który skłądaja sie dane dotyczące środowiska GUI

beep() - odtwarza dzwięk
